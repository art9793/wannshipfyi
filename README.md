# WannaShip.fyi

50 validated startup ideas with step-by-step playbooks — from idea to first dollar.

**[wannaship.fyi](https://wannaship.fyi)**

## What is this?

A curated Notion database of research-backed startup ideas, each with a complete playbook covering market validation, MVP scope, monetization strategy, and growth tactics. Built for developers and makers who want to skip the "what should I build?" phase and start shipping.

## What's in the database?

- 50 ideas across SaaS, marketplaces, developer tools, and more
- Pre-built Notion views: Quick Wins, High Revenue Potential, Solo-Friendly
- Each playbook includes: problem analysis, target audience, revenue model, tech stack suggestions, and first 30 days action plan

## Tech stack

Static site hosted on GitHub Pages. No build step, no dependencies, no framework.

## Blog

SEO articles on micro-SaaS, side projects, and startup validation — all at [wannaship.fyi/blog](https://wannaship.fyi/blog).
